    The template code provided in Makerchip by default, and simple Verilog templates.

    SystemVerilog Testbench

    To keep things simple, most Makerchip examples include testbench content (stimulus and checking) within the logic of the example. In real-world usage, TL-Verilog would define hardware logic within a module, and an outer testbench module would instantiate this module to test it. This example illustrates this.

    Combinational Logic Examples

    A bit-level incrementer, using hierarchy to replicate the bit-slice.

    Fibonacci Sequence

    The Fibonacci Sequence is a sequence of whole numbers starting with two 1s, where each subsequent value in the sequence is the sum of the previous two values.

    Pythagorean Theorem Pipeline

    Pythagoras's Theorem provides the length c below, given a and b.

    c = sqrt(a2 + b2)
    Pythagorean Theorem

    You can use plain old Verilog in Makerchip if you really want to.

    Incrementer

    A bit-level incrementer, using behavioral hierarchy to replicate bit slices.

    Ripple-Carry Adder

    A bit-level ripple-carry adder implemented as a macro.

    FPGA Multiplier

    Some FPGAs contain fixed-width multiply units. This example shows how the fixed-width multipliers can be used to implement a wider multiply by summing partial-products in a pipelined calculation. It is based on a VHDL example from Surf VHDL.

    Long Division

    This example performs hexidecimal long division. Pipeline depth is configurable.

    Finite State Machine

    This example implements an elevator controller in SystemVerilog and TL-Verilog. It suggests different coding styles between the two. This example is useful for understanding code conversion practices.

    Arrays

    A simple array macro, one holding transactions, and a 2-D transaction array implemented several different ways.

    Backpressured Pipeline Macro

    This example illustrates the use of an M4 macro that instantiates a backpressured pipeline. In the first case, the pipeline is empty; in the second, a Pythagorean Theorem example is implemented within the backpressured pipeline.

    Speculation Macro

    This examples shows the use of a macro that provides the control flow for a speculative computation, providing a speculative result a cycle earlier than the full computation. It illustrates how, with TL-Verilog, bug-prone control logic can be implemented in library components. This macro is used to implement a variable-delay adder and can provide the speculative flow for other computations just as easily.

    CPUs

    A tiny non-pipelined educational CPU in < 200 lines of code.

    A parameterized pipelined version of this mini-CPU in ~250 lines of code.

    WARP-V, is a highly-adaptable TL-Verilog CPU generator, with flexible pipeline staging and support for multiple instruction-set architectures (RISC-V, MIPS, ...). WARP-V has been implemented as a cloud FPGA microservice using 1st CLaaS. The WARP-V project is living and breathing in github and even has a configurator at warp-v.org .

    This toy MIPS CPU example comes from a course at ASU.

    Hardware-Accelerated Web Applications on Cloud FPGAs

    TL-Verilog is utilized in 1st CLaaS, a framework for creating FPGA-accelerated web and cloud applications . These fractal flythroughs were generated by an Amazon F1 Cloud FPGA programmed with TL-Verilog logic, and you can fly for yourself at fractalvalley.net.

    This accelerator was implemented in Makerchip, and debugged with VIZ.

    Visit the 1st CLaaS repository to learn how you can deploy your Makerchip designs in the cloud.

    Imported SystemVerilog Designs

    These examples illustrate how to import SystemVerilog designs from external repositories and built them in Makerchip.

    Olof Kindgren's SERV core is an award-winning bit-serial RISC-V CPU.

    BaseJump STL is a library of SystemVerilog components from the Bespoke Silicon Group . This example imports the library and uses a FIFO.
